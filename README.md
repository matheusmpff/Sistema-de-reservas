# Sistema-de-reservas

## Objetivos

O sistema está sendo desenvolvido para uma pousada localizada na cidade de São Miguel do Gostoso. O objetivo principal do sistema é automatizar certas ações necessárias para conclusão
de uma reserva nesse estabelecimento. Para isso será desenvolvido um site em que o usuário terá acesso a funcionalidades como:

+ Acesso aos tipos de quarto atualmente disponíveis
+ Auxilio no passo a passo até a etapa da conclusão do pagamento em que é direcionado a um funcionário do estabelecimento
+ Ter uma conta atrelada ao site
+ Pode responder um formulário de feedback com upload de imagens

## Parte 1
Durante a primeira parte do desenvolvimento, implementamos as paginas do Front-end do site buscando uma distribuição e ordenamento dos elementos de forma harmonica. O principal objetivo com um design responsivo e atrativo é fornecer uma boa experiência de usuário de forma a levar a conversão de cenda para a pousada.

## Como foi desenvolido

Para a immplementação das páginas do Front-end foi utilizado:

- HTML
- CSS
- BOOTSTRAP
- JAVASCRIPT
